## Основные термины

| Английский | Русский | Контекст |
|-----------|---------|----------|
| The Nation | Нация | Государство (всегда с большой буквы) |

| The Office | Управление | Спецслужба, ведомство |
| Operator | Оператор | Сотрудник Orwell (игрок) |
| Handler | Куратор | Руководитель оператора |
| Adviser | Советник | Куратор (альт. название в игре) |

| Reader | Ридер | Вкладка чтения документов |
| Listener | Перехват | Вкладка прослушки |
| Bookmarks | Закладки | Вкладка закладок |
| Target person | Цель / Объект наблюдения | Человек под слежкой |
